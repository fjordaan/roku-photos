# Roadmap

Development stages, starting with core functionality, moving on to nice-to-have features.

## Stage 1: Hello World

- Demonstrate that we can install an app on the Roku
- Demonstrate that we can read and display photos from the NAS
- Demonstrate that we can detect all button presses on Roku remote

## Stage 2: Basic screensaver without menu

- Create server app to run on NAS
- Build an index of all photos in Library
- Display them as a slideshow in random order
- Ability to go back to previously displayed photos, and forward
- Buttons: Left / Right and Play / Pause

## Stage 3: Screensaver menus: Core options

- Implement Exit menu
- Implement Photos menu
  - All photos
  - This folder only
- Implement Order menu
  - Random
  - A-Z by filename
- Implement Settings menu
  - Slide duration
  - Looping / Stop at end
- Buttons (in menus): Up / Down / Left / Right / OK / Back
- Buttons (in slideshow): Back 10, Forward 10, Down (Display photo properties)

## Stage 4: More options using dynamic playlists

- Index capabilities that power dynamic playlist generation
- Photos menu
  - This Day / Month / Year
  - With tag
- Order menu
  - Chronological
- Settings menu
  - Reset all to defaults
 
## Stage 5: Nice-to-haves

- Automatic activation as Roku native screensaver
- Video support
- Picture settings (fit / fill / crop)
- Thumbnails (Up button)
- Select Library / Custom playlists
- Photo blocklist
- Browse folders
